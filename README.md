#  Tasqly — Minimal Todo App

Tasqly is a  minimal Todo Application built using React and Tailwind CSS.  
It helps users manage daily tasks in a simple and organized way with a soft modern UI.

---

#  Features

-  Add Todos
-  Mark Tasks as Completed
-  Delete Todos
-  Edit Existing Todos
-  Local Storage Support
-  Show / Hide Finished Tasks
-  Responsive Design
-  Calm Minimal UI
-  Galaxy Background Effect

---

#  Built With

- React.js
- Tailwind CSS
- UUID
- React Icons
- Font Awesome

#  Installation

##  Clone Repository

```bash
git clone https://github.com/eshmalarshad/Tasqly.git
```

---

##  Go to Project Folder

```bash
cd tasqly
```

---

##  Install Dependencies

```bash
npm install
```

---

##  Start Development Server

```bash
npm run dev
```

---

#  Dependencies Used

Install these if needed:

```bash
npm install uuid
npm install react-icons
npm install @fortawesome/react-fontawesome
npm install @fortawesome/free-regular-svg-icons
```
